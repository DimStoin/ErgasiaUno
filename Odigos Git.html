Εντόλες GIT

  1. Ενεργοποιούμε τον φάκελο ως νέο project $ git init
  2. Το $ git status μας λέει αν αναγνωρίζονται ή οχι τα αρχεία του φακέλου.
  3. Για να προσθέσουμε όλα τα αρχεία του φακέλου γράφουμε $ git add.
  4. Χρησιμοποιούμε το $ git commit -m " " για να κάνουμε commit και να προσθέσουμε ένα μήνυμα.
  5. Αν θέλουμε να δούμε ποιά commit έχουν γίνει χρησιμοποιούμε το $ git log
  6. $ git remote add origin αντιστοιχεί το local repository με το online
  7. $ git push -u origin master ανεβάζει το περιεχόμενο στο origin που ορίσαμε πρίν.
  8. $ git pull κατεβάζει όλα τα αρχεία του repository.
